Nishio Tweet Manager: Windows, Linux, Macで動作するTwitterクライアント

1. 本ソフトウェアについて

Nishio Tweet ManagerはWindows, Linux, Macとあらゆる環境で動作するTwitterクライアントです．
本ソフトウェアは以下の特徴を持っています．

 * Windows, Linux, Macとあらゆる環境で動作
 * Streaming APIを利用したツイート取得
 * タイムラインをログ情報として保存
 * 気になったワードを検索し，別タブにて表示できる
 * 気になったワードに関連するハッシュタグを検索できる 
 * 保存したタイムラインをCSV等で出力
 * 過去ログを簡単に閲覧可能

3.Javaについて

本ソフトウェアを動作させるためには，Sun Java version6以上が必要です．
Javaは　http://www.java.com/ja/download/ からダウンロードが可能です．

4. プログラム起動方法

Windows上でのプログラム実行方法
 -NishioTweetManager.exe を起動してください

Mac上でのプログラム実行方法
 -NishioTweetManager.appを起動してください

Linux上でのプログラム実行方法
 -ターミナル上で java -jar NishioTweetManager.jar を入力し，プログラムを実行してください．

5. 利用方法

利用方法の詳細については http://sourceforge.jp/projects/nt-manager/wiki/howtouse をご覧ください．
